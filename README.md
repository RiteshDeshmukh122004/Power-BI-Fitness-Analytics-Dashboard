# Project Overview
The Fitness Analytic Dashboard is an interactive data visualization project designed to analyze fitness activities, membership data, finances, and health metrics.
This dashboard helps gym owners, trainers, and users track performance, revenue, expenses, and personal fitness indicators like BMI and calorie requirements.

# Objectives
- Visualize gym financial performance (Revenue, Expenses, Profit)
- Track client memberships and their status (Active / Expired)
- Analyze members by age, gender, and membership type
- Calculate BMI, BMR, TDEE, and calorie requirements
- Provide interactive filters and dashboards for insights
  
# Dashboard Features
### 1. Home 
- Total Clients
- Total Trainers
- Revenue & Expenses summary
- Theme selection (Orange, Blue, Purple, Green)
 
### 2. Overall Analytics
- Monthly Revenue, Expenses, and Profit chart
- Membership distribution (Platinum, Gold, Silver)
- Client membership status (Active / Expired)
- Monthly new members trend
  
### 3. Fitness Calculator
- BMI (Body Mass Index) 
- BMR (Basal Metabolic Rate)
- TDEE (Total Daily Energy Expenditure)
- Maintenance Calories
- Weight Loss & Extreme Weight Loss Calories
- Age, Height, Weight sliders
- Gender selection
  
### 4. Members Analytics
- Members by Age & Gender
- Members by Age & Status
- Detailed Member Information Table
 1 Name
 2 Gender
 3 Age
 4 Goal (Weight Loss / Muscle Gain / Maintenance)
 5 Membership Type
 6 BMI
